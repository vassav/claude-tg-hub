# claude-tg-hub

Единый Telegram-бот (сервис) для управления сессиями **Claude Code**: список активных
сессий, двустороннее взаимодействие, уведомления и подтверждение действий (аппрувы) —
для консольных (CLI) и, по возможности, VSCode-сессий.

> 🚧 Статус: **этап дизайна**. Кода пока нет. Дизайн и решения — в
> [`docs/specs/2026-06-05-claude-tg-hub-design.md`](docs/specs/2026-06-05-claude-tg-hub-design.md).

## Что хочется (требования)

1. Список всех активных сессий Claude Code.
2. Взаимодействие с любой сессией: сообщение/команда, статус, ответ.
3. Уведомления.
4. Подтверждение действий: аппрувы/селекторы.
5. Поддержка CLI-сессий и сессий расширения VSCode.

## Ключевой вывод исследования

Готового решения, закрывающего всё (особенно **панель VSCode + аппрувы + мульти-сессии**),
нет. Главное ограничение — панель-расширение VSCode не отдаёт точки интеграции для входящего
потока и аппрувов (ограничения самого Claude Code). Детали и матрица возможностей — в спеке.

## Стек

- Node.js + TypeScript
- Telegram: grammy (или telegraf)
- Интеграция с Claude Code: hooks (`~/.claude/settings.json`) + Claude Agent SDK + чтение
  `~/.claude/projects/**/*.jsonl`
